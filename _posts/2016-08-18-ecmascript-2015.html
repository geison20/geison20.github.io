A nova versão do JavaScript | ECMAScript (ES6) implementa novas features na linguagem, hoje vamos falar um pouco de let e const.

let:

O let permite que você declare uma variavel limitando seu escopo interno diferente de var que usa o escopo global.

exemplo de let

{% highlight javascript linenos %} #!javascript/app.js var a = 5; if (a === 5) { console.log(a); let a = 10; } console.log(a); {% endhighlight %}

Nesse exemplo simples eu declaro uma variável a = 5 inicialmente, logo após verifico se a === 5 (inclusive o tipo), entrando no bloco condicional (true) eu atribuo agora a váriavel let a = 10, faço um console.log(a) dentro do bloco e outro fora, isso ocorre por que let está declarada dentro do escopo de bloco.

conts:

Esta declaração cria uma constante que pode pertencer ao escopo global ou local de uma função, o valor de uma constante não pode ser alterado via re-atribuição, e uma constante não pode ser redeclarada. É obrigatório que um valor seja atribuido à constante em sua declaração.

exemplo de const

{% highlight javascript linenos %} #!javascript/app.js const cpf = 123456789; console.log(cpf); cpf = 987654321; var cpf = 987654321; console.log(cpf); {% endhighlight %}
